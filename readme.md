> ## Instalation

### Yarn

```
  yarn global add xcra
```

### Npm

```
  npm --global i xcra
```



## Create ReactJS Project
```
  xcra react --name <NAME PROCJET>
```

*Await process ending*


## Create React Native Project
```
  xcra react-native --name <NAME PROCJET>
```

*Await process ending*
